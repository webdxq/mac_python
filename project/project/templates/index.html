Disulfide Bond Prediction

Provide candidate disulfide bonds for you


  • This website tool will give you some amino acid pairs which are have potential to form a disulfide bond after mutated to CYS.
  • The result is a list of amino acid pairs with the probability of form a disulfide bonds.
  • To use this predicition tool, you only need to input a local pdb file or input a pdb id. Then wait the server give the result.
